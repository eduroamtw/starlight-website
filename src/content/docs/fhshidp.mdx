# 安裝步驟
1. 下載下面的設定檔 (二擇一)。
2. 依據 geteduroam 使用教學 ([**點我前往**](/guides/geteduroam)) 中的步驟 (**從第二步開始**)，匯入剛才下載的安裝檔。

# 帳號來源
目前有下列兩個帳號來源可以使用：
- 鳳新 Google 帳號
- 教育雲端帳號

我們推薦您使用鳳新 Google 帳號認證，節省您申辦教育雲帳號的時間。  
請您依據您帳號來源，選擇你要安裝的設定檔 (eduroam 以及 Fhsh-Eduroam 皆會同時安裝)。
