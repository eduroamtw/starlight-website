# 使用說明
目前有下列兩個帳號來源可以使用：
- 鳳新 Google 帳號
- 教育雲。

我們推薦您使用鳳新 Google 帳號認證，節省您申辦教育雲帳號的時間。  
接入點部分，目前我們提供校內的 Fhsh-Eduroam 接入點以及全球通用的 eduroam 接入點設定，
請您依據您帳號來源，選擇正確的兩個設定檔安裝。  
註：由於可用性問題以及嚴重的資安風險問題，我們已停止支援由高雄資教維護 (高雄部分高中、國中小) 的魔改版 eduroam。
除非高雄資教將其維運的 eduroam 依照 RFC7593 的規範設定，否則您將無法在高雄資教的 SP 連線，敬請見諒。
