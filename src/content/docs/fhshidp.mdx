# 使用說明
## 應用程式
使用我們的設定檔，您需要先下載安裝 geteduroam 應用程式 ([**點我下載**](http://eduroam.app/))。  
應用程式使用教學請參見 geteduroam 使用教學 ([**點我前往**](/guides/geteduroam))。

## 帳號來源
目前有下列兩個帳號來源可以使用：
- 鳳新 Google 帳號
- 教育雲端帳號

我們推薦您使用鳳新 Google 帳號認證，節省您申辦教育雲帳號的時間。  
接入點部分，目前我們提供全球通用的 eduroam 接入點以及校內的 Fhsh-Eduroam 接入點設定，  
請您依據您帳號來源，選擇你要安裝的設定檔 (eduroam 以及 Fhsh-Eduroam 皆會同時安裝)。  

# 安裝步驟
1. 先安裝 geteduroam 應用程式 ([**點我下載**](http://eduroam.app/))
2. 下載下面的設定檔 (二擇一)。
3. 按 geteduroam 使用教學 ([**點我前往**](/guides/geteduroam)) 中的步驟，匯入剛才下載的安裝檔。
