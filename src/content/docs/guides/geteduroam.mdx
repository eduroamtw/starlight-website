**如何使用 geteduroam 以及匯入設定檔。**

import { Tabs, TabItem } from '@astrojs/starlight/components';

## 系統支援度
- 可使用的系統
    - Windows
    - Linux
    - Android
    - iOS
- 不可使用的系統
    - macOS

## **⬇️請選擇你的系統⬇️**
<Tabs>
    <TabItem label="Windows" icon="seti:windows">
    ## Windows
    0. 下載各校設定檔 [API](https://download.isli.me/) [GitHub](https://github.com/eduroamtw/geteduroam_tw/tree/main/Profile)
    ![win01](geteduroam/win01.png)
    1. 下載 geteduroam 主程式 [**一般電腦 (amd64)**](https://dl.eduroam.app/windows/amd64/geteduroam.exe) [**arm64**](https://dl.eduroam.app/windows/arm64/geteduroam.exe)
    ![win02](geteduroam/win02.png)
    2. 新增一個資料夾，名稱任意。
    3. 將剛剛下載的設定檔與 geteduroam 主程式放進去剛才新增的資料夾。
    4. 打開 geteduroam 主程式。
    ![win03](geteduroam/win03.png)
    5. 確認設定檔資訊後，按 Next
    ![win04](geteduroam/win04.png)
    > 如果跳出附圖的畫面，則依序點選 
    > Install
    > 是
    > Next
    > 資安警告：請勿安裝任何未知來源的設定檔。
    ![win05](geteduroam/win05.png)
    ![win06](geteduroam/win06.png)
    ![win07](geteduroam/win07.png)
    6. 輸入你單位帳密
    7. 按 Connect
    ![win08](geteduroam/win08.png)
    8. 出現 Everything is set up correctly! 後，就可以快樂上網了。
    ![win09](geteduroam/win09.png)

    </TabItem>
    <TabItem label="Android" icon="phone">
    ## Android
    0. 下載設定檔
    1. 下載 geteduroam 程式 [**Google Play**](https://play.google.com/store/apps/details?id=app.eduroam.geteduroam&pcampaignid=web_share)
    2. 打開他
    ![android01](geteduroam/android01.png)
    3. 長按右上搜尋鍵
    4. 點選 Open .eapconfig file
    ![android02](geteduroam/android02.png)
    5. 打開剛剛下載的設定檔檔案
    ![android03](geteduroam/android03.png)
    6. 輸入你的帳號密碼，之後按下 Log in
    ![android04](geteduroam/android04.png)
    7. 在跳出來是否儲存網路的選單中，選擇
    > Save
    > 確認
    > 儲存
    等字眼的選項
    ![android05](geteduroam/android05.png)

    </TabItem>
    <TabItem label="iOS" icon="apple">
    ## iOS
    0. 下載設定檔
    1. 下載 geteduroam 程式 [**App Store**](https://apps.apple.com/app/geteduroam/id1504076137)
    2. 打開他
    ![ios01](geteduroam/ios01.png)
    3. 長按右上搜尋鍵
    4. 點選 Open .eapconfig file
    ![ios02](geteduroam/ios02.png)
    5. 打開剛剛下載的設定檔檔案
    ![ios03](geteduroam/ios03.png)
    6. 輸入你的帳號密碼，之後按下 Log in
    ![ios04](geteduroam/ios04.png)

    </TabItem>
